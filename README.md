# Smart Land Copilot v7.0

منصة ذكية لإدارة الأراضي الاستثمارية في مصر — تجمع بين الاستشعار عن بعد، الذكاء الاصطناعي، والتكامل مع الخدمات الحكومية لتوفير تجربة متكاملة للمستثمرين وملاك الأراضي والوسطاء.

## الميزات الرئيسية

- **المزاد والمنصة التجارية** — مزادات مباشرة مع عرض شفاف للرسوم والضرائب
- **محرك المطابقة الذكي** — يطابق المستثمرين بالأراضي المناسبة باستخدام GLM
- **نظام الوسطاء** — إدارة الوسطاء، العمولات، والتعيينات
- **مجتمع البحث عن الأراضي** — Scout system لتأمين فرص استثمارية جديدة
- **التحقق من المستندات** — التحقق من وثائق الملكية باستخدام الـ GPS
- **التحليل البيئي والبنية التحتية** — تقييم الكثافة الخضراء والمرافق
- **التقارير الاستشارية بالذكاء الاصطناعي** — تقارير جدوى شاملة باستخدام LLM

## المتطلبات

- Python 3.12+
- PostgreSQL (اختياري، يمكن استخدام SQLite للتطوير)
- Ollama (اختياري، للنماذج المحلية)

## التثبيت والتشغيل

```bash
# تثبيت المتطلبات
pip install -r requirements.txt
pip install -r requirements-dev.txt

# تشغيل التطبيق
streamlit run app.py
```
